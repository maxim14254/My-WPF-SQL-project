Данная программа высчитывает количество баллов за выполнение физических упражнений с учетом требований утвержденных приказом Министра обороны Российской Федерации № 200. от 21 апреля 2009 г. Оценка выставляется по 100 бальной шкале (за каждое упражнение) на основании полученных результатов по трем упражнениям (сила, быстрота, выносливость).
